# DRONE - STM32F411


# Pin Connection
############### MPU6050 ###############
- VCC -> 3.3V
- GND -> GND
- SCL -> PB6 (I2C1)
- SDA -> PB7
############### ESC ###############
- ESC1 -> TIM (PA0)
- ESC2 -> PA1
- ESC3 -> PA2
- ESC4 -> PA3
############### ESP32 ###############
- TX <-> RX (UART)
- RX <-> TX
- GND
